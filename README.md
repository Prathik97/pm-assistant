# PM Assistant — AI PRD Co-pilot

A working AI tool that evaluates whether your problem actually needs AI, 
then generates a complete 10-section PRD.

## What it does
- Evaluates any problem statement against a 4-question framework
- Determines if the solution should be AI, rules-based, or hybrid
- Asks 5 targeted clarifying questions
- Shows a verification brief of every assumption before generating
- Generates a complete 10-section production-ready PRD

## Built with
Claude API · Vanilla JS · localStorage · No backend · No frameworks
